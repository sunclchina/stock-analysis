### 三、宏观政策面影响分析（个股/批量通用，模型自动分析）
结合A股政策驱动的核心特征，重点跟踪四大政策维度，分析对个股/所属行业的直接及间接影响，判断政策催化力度与持续性：
1. 货币政策：跟踪LPR、MLF调整（利率变动/投放规模）及存款准备金率变化，分析流动性宽松/收紧对个股估值的影响——如降息降准释放流动性，利好成长股、高估值个股；流动性收紧则优先规避高杠杆、高估值标的。模型自动结合当前最新货币政策展开详细分析，明确影响方向及程度。
2. 财政政策：关注专项债发行、减税降费、产业补贴等政策，重点分析财政发力方向与个股/所属行业的契合度（如基建补贴利好基建板块个股，消费补贴带动消费类个股）。模型自动结合当前最新财政政策，分析对标的的具体影响。
3. 产业政策：聚焦所属行业的国家规划、行业指导意见，如AI、新能源、半导体等赛道的政策扶持，判断政策对行业景气度的提升作用，以及个股在产业链中的受益程度；同时关注监管动态（IPO节奏、减持新规）对个股的影响。模型自动结合标的所属行业，分析相关产业政策的影响。
4. 外部消息：跟踪全球主要经济体政策、国际经贸环境、外围市场动态，分析外部不确定性对个股（尤其是出口型、外资重仓个股）的传导影响，判断短期股价波动风险。模型自动结合当前外部环境，分析潜在影响及风险。
影响总结：模型自动明确政策整体影响（利好/利空/中性）、影响程度（强/中/弱）、持续性（长期/中期/短期），并明确重点关注的核心政策落地情况。
### 四、行业景气度分析（个股/批量通用，模型自动分析）
遵循“选对行业比选对个股更重要”的A股实战逻辑，跟踪行业核心指标，识别高景气赛道，判断个股在行业中的竞争力与成长空间：
1. 行业所处周期：模型自动判断标的所属行业当前所处周期（导入期/成长期/成熟期/衰退期），并结合周期特征分析估值逻辑（成长期行业享受高估值溢价，成熟期行业重点看分红与成本控制，衰退期行业尽量规避）。
2. 业绩与行业数据：模型自动跟踪行业最新营收增速、利润增速、订单情况等核心数据，结合个股业绩预告/年报，分析个股业绩与行业景气度的匹配度；重点关注行业数据超预期的领域，挖掘业绩驱动型标的，并给出具体分析。
3. 产业链动态：模型自动梳理个股在产业链中的位置（上游/中游/下游），跟踪原材料价格、下游需求变化，分析上下游波动对个股盈利的影响；关注行业竞争格局（CR5集中度、竞争壁垒），判断龙头个股的份额优势与盈利稳定性，给出详细分析。
4. 高景气赛道识别：模型自动结合政策支持与行业数据，判断所属赛道是否为当前高景气赛道（如AI算力、新能源、高端制造），分析个股在高景气赛道中的核心竞争力（技术、产能、市场份额），明确竞争优势。
景气度结论：模型自动明确所属行业景气度（高/中/低）、行业竞争格局（集中/分散）、个股在行业中地位（龙头/中游/尾部）及中长期成长空间（大/中/小），形成完整结论。
