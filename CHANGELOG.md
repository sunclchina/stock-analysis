# 更新日志

## [1.0.0] - 2026-05-06

### 新增
- 系统仪表盘（M01）：大盘概览、行情快速查看、系统状态
- 实时行情（M02）：股票行情表格、K线图表、盘前提示卡片
- 智能预警（M03）：7大预警模块（价格/涨跌/趋势/共振/财务/突发/风险）、综合决策矩阵、监控面板
- 智能选股（M04）：固定规则选股（8套模板）、自定义条件选股
- 智能分析（M05）：大盘复盘、个股深度分析、批量分析
- 系统配置（M06）：自选股管理、监控池管理、数据源切换、模板管理、用户偏好
- 资产组合管理（M07）：虚拟持仓、自动交易引擎
- 用户体系：JWT认证、角色权限（管理员/普通用户）、用户管理
- 帮助中心：文档分类、全文搜索、用户反馈、联系我们
- 数据源系统：通达信本地、东方财富、新浪财经、Baostock+
- WebSocket 实时推送

### 技术栈
- 后端：Python 3.12+ / FastAPI / SQLAlchemy / APScheduler
- 前端：React 18 / TypeScript / Ant Design 5 / Vite / Zustand / ECharts
- 数据库：SQLite（开发）/ 可迁移 MySQL（生产）

## 版本号格式

版本号遵循 [语义化版本 2.0.0](https://semver.org/lang/zh-CN/)：

```
主版本号.次版本号.修订号
```

- **主版本号**：不兼容的 API 或架构变更
- **次版本号**：向下兼容的功能新增
- **修订号**：向下兼容的问题修复

## 升级说明

开发环境保留不变。升级步骤：

1. 从 Git 拉取最新代码
2. 后端：更新依赖 `pip install -r requirements.txt --upgrade`
3. 前端：更新依赖 `npm install`
4. 执行构建：`deploy\build_prod.ps1`（Windows）或 `docker compose build`（Docker）
5. 重启服务

详细升级指南见 `UPGRADE.md`。
